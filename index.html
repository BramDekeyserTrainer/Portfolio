Opbouw

Deze portfolio is opgebouwd uit enkele secties. Ik zal kort wat uitleg geven over iedere sectie om de opbouw te verduidelijken.   

  • De eerste sectie waren de twee slides. Hier staat een knop die linkt naar Mijn Profiel. Deze staat boven deze sectie.
  • Deze sectie zelf is de tweede sectie. In deze sectie leg ik uit hoe de website is opgebouwd.
  • Deze sectie bevat ook een link naar de Navigatie Opdracht die we eerder deze module hebben moeten maken.
  • In de volgende sectie zal ik twee werkjes tonen die ik in de module Interface Design heb gemaakt.
  • Nadien zal ik kort wat laten zien over wat ik maakte bij Grafische Vormgeving.
  • ook zal ik een paar Photoshop werkjes tonen.
  • Ten slotte is er ook een sectie met enkele buttons van sites die ik al eerder heb gemaakt.

Ziet u deze knop hieronder? Deze knop is een link die u zal leiden naar een eerder door mij gemaakte opdracht. Herinneren jullie nog die tijd dat we nog alleen maar een simpele navigatie moesten maken zonder al te veel tralalie en tralala? Dit is mijn versie van de opdracht van toen!
